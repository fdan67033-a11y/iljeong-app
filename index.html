연결 설정

이 앱이 GitHub 비공개 저장소의 일정을 읽고/추가하려면 토큰 한 번이 필요합니다.

  1. github.com → 우측상단 프로필 → Settings
  2. 맨 아래 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  3. 이름 아무거나 · Repository access: Only select → iljeong-sync
  4. Permissions → Repository → Contents: Read and write
  5. Generate → 나온 토큰(github_pat_…) 복사 → 아래 붙여넣기
토큰은 이 폰에만 저장됩니다(다른 곳으로 안 보냄).
